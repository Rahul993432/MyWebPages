C - The Programming Language

C is a High Level Computer Programming Language developed by Dennis Ritchie. It is also Known as Middle Level Laguage. It is a Compiler based language. It is case sensitive language. We can develop any type of application using it.

WORD:-

There are two types of words in C:-

  1. Keyword

  2. Identifier

1. Keyword:- The words reserved by compiler. C has 32 keywords. Keywords are always writen in lower case.

Ex:-int, float, double, if, else, for, do, while, switch, break, continue etc.

Note:-C is a case sensitive language.

Case Sensitive:-Both case are different symbol.

Case Insensitive:-Both case are same symbol.

2. Identifier:- The words used to identify any entity.

Rules for naming identifier:-

It should consist of alphabet(Upper or Lower), Digits(0-9) or underscore( _ ).

FILES:-

Secoundary name is also known as Extension.

Ex:- rahul.txt text file, .c C source file, .cpp C++ source file, .java Java source file, .py Python source code, .js JavaScript source code, .css Cascoding styple sheets code, .bas Qbasic source code, .html or .htm HTML source code

Executing a C program:-

STEP 1: Write code using C in a file and save it with any primary name having extension (.c). The code written in any language is called SOURCE CODE.

STEP 2: Send you source code to c COMPILER. Compiler will check whole code for syntax error, if any error compiler will give the error list. If no syntax error found compiler will translate the code and give a new code known as EXECUTABLE CODE. The executable code is saved with any primary name having extension(.exe).

Variables :-

In program variable are space in memory (RAM) that can store data. Every variable as unique name.

In C every variable must be declared before use.

Variable Declaration Syntax:-

DataType VariableName;

DataType:-Any valid C data type.

C has Four fundamental data type.

  1. int -> keyword -> Represents integer value ( Numberic value without decimal point).

  2. float -> keyword -> Represents Single precision real/float value ( Numeric value with decimal point).

  3. double -> keyword -> Represents double precision real/float value.

  4. char -> keyword -> Represents a single character value.

VariableName -> any valid C identifier.

ex:-

int roll; Here roll is an integer variable. It an store an integer value.

float marks; Here marks ia a float variable. It can store a character.

char section; Here section is a character variable. It can store a character.

Assigning value to variable :-

Here can be a literal/expression or function call.

Ex:- roll=10;       //10 is assigned to roll.

marks=78.90;     //78.90 is assigned to marks.

1. Arithmetic Operator :-

There are 5 arithmetic operators in C :-

  1.          + ( Addition )

  2.          - ( Subtraction )

  3.          * ( multiplication )

  4.          / ( Division )

  5.          % ( Modular Division )

All are Binary Operators.

/ Perform division and gives QUOTENT as result.

% perform division and gives REMINDER as result.

The expression having arithmetic operator is called arithmetic expression. Arithmetic expression always yields a Numeric Value.

The operator +, -, *, and / can work with any numeric type operand i.e. int or float but % both operands must be integer type.

% can't work with float value.

Ex :-

7 % 3 -> Correct (both operand are of int type)

7.0 % 3 -> Error

7 % 3.0 -> Error

pow( ):-used to find power of any number. if pow( ) is used in program then "math.h" must be include in program.

Syntax :-

pow( BaseValue,PowerValue )

This function will accept two values as argument. the first value will be treated base and second will be treated power and after operation it will return a numeric value.

Ex :-

        x=pow(10,3);

pow( ) will find the 10 to the power 3 and the returned value will be stored in variable x. that is after completing this statement the value of x will be 1000.

The arguments of pow( ) can be literal or variable.

int a,b,c;

a=2;

c=10;

b=pow(a,c); here pow( ) will calculate 2 to the returned value will be stored in b i.e. 1024.
