Computer Second Generation :-

The Second generation computer used Transistor in CPU, I/O processer and other electronic component.The Superiority of Transistorover vacuume tube made the computer smaller in size, Faster and more energy efficient.

The importent features Second generation computer are :-

  • Transistor were use in CPU, I/O processer and other electronic component.
  • Magnetic disk and magnetic tape were used as a secondry memory.
  • Magnetic ferrite core was used as main memory.
  • Machine independent high level language like FORTRAN was used as programming language.
  • Hardwere for floating point arithmetic was developed.
  • I/O processor was developed to supervise and control input output operation.
  • Index register wasv developed which increased the floxibility in programming.
  • Use of transister made the computer smaller, faster and reliable.
