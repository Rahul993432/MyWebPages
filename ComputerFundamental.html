COMPUTER FUNDAMENTAL :-

Computer :- Computer is an electronic devise the accept the data as an input, process the data according to the instraction and gives and output.

According to the definition computer performus the following task:-

  1. It accepts the data as an input.
  2. Process the data according to the instraction.
  3. Gives result as an o/p. or store the result in same memory location.

Characteristies of computer :-

  1. Speed :-Computer can perform logical and mathematical calculation and much higher speed then humen. If perfoumes millions of calculation in a second. To days speed of cumputer is of the order of micro speed, nano speed and pico second.
  2. Accuray :-Computer perform mathematical calculation with 100% accuracy. Their may be some error due to inconsistancy or inaccuracy of data.
  3. Automation :-Computer performs mathematical calculation automaticaly without any intervantion of humen.
  4. Storeg :-Computer can store large amount of data. it can store the data in any form.
  5. Diligence :-Computer can performs a number of task without any fatigue or lack of consitration.
  6. Multitasking :-Computer can performs multipul task and the same time.
  7. Varsatility :-Computer can performs different kinds of work with same accuracy and efficiency.
  8. No Feeling :-Computer have no feeling and emotion. It can works continusly according to the instraction provided.
  9. No IQ :-Computer have no IQ it is a dumb machine with out user it works only according to the instraction provided by the user. it can not corret or modify the data it salfe if some inconsistancy or inaccurate data is provided.
