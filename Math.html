Important Algebraic Formula :-

  • (a + b)2 = a2 + 2ab + b2

  • (a - b)2 = a2 - 2ab + b2

  • a2 - b2 = (a + b) (a - b)

  • a2 + b2 = (a + b)2 - 2ab

  • a2 + b2 = (a - b)2 + 2ab

  • a3 + b3 = (a + b)3 - 3ab(a + b)

  • a3 - b3 = (a - b)3 + 3ab(a - b)

  • 2(a2 + b2) = (a + b)2 + (a - b)2

  • (a + b)2- (a - b)2 = 4ab

  • a4 + b4 = (a + b)(a - b)[(a + b)2 - 2ab]

  • (a - b)2 = (a + b)2 - 4ab

  • (a + b)2 = (a - b)2 + 4ab

  • a4 + b4 = [(a + b)2 - 2ab]2 - 2(ab)2

  • (a + b + c)2 = a2 + b2 + c2 + 2ab + 2bc + 2ca

  • (a + b - c)2 = a2 + b2 + c2 + 2ab - 2bc - 2ca

  • (a - b - c)2 = a2 + b2 + c2 - 2ab - 2bc - 2ca

  • a3 + b3 + c3 - 3abc = (a + b + c) (a2 + b2 + c2 - ab - bc - ca)

  • a4 + a2 b2 + b4 = (a2 + ab + b2) (a2 - ab + b2)

  • a4 + a2 + 1 = (a2 + a + 1) (a2 - a + 1)

    • if a + b + c = 0 then a3 + b3 + c3 = 3abc

  • a8 - b8 = (a4 + b4) (a + b) (a - b)

Important Trigonometric Formula :-

  • sin(A+B) = SinAcosB + cosAsinB
  • sin(A-B) = sinAcosB - cosAsinB
  • cos(A+B) = cosAcosB - sinAsinB
  • cos(A-B) = cosAcosB + sinAsinB

  • tan(A+B) = tanA + tanB / 1 - tanAtanB
  • tan(A-B) = tanA - tanB / 1 + tanAtanB
  • cot(A+B) = cotAcotB - 1 / cotA + cotB
  • cot(A-B) = cotAcotB - 1 / cotB - cotA

  • sin2A =
    1. 2sinAcosA
    2. 2tanA / 1 + tan2A
  • cos2A =
    1. cos2A - sin2A
    2. 1 - sin2A
    3. 2cos2A - 1
    4. 1 - tan2A / 1 + tan2A

  • tan2A = 2tanA / 1 - tan2A
  • sin3A = 3sinA - 4sin3A
  • cos3A = 4cos3A - 3cosA
  • tan3A = 3tanA - tan3A / 1 - 3tan2A

  • sin(-A) = - sinA
  • cos(-A) = cosA

  • sin2A + cos2A = 1

  • 1 + tan2A = sec2A
  • 1 + cot2A = cosec2A
  • 1 - cos2A = 2sin2A
  • 1 + cos2A = 2cos2A
  • 1 - sin2A = (cosA - sinA)2
  • 1 + sin2A = (cosA + sinA)2
  • 1 + tanA / 1 - tanA = tan(pi/4 + A)
  • 1 - tanA / 1 + tanA = tan(pi/4 - A)

  • sin2A - sin2B = sin(A + B) sin(A - B)
  • cos2A - sin2B = cos(A + B) cos(A - B)
  • cos2A - cos2B = sin(A + B) sin(A - B)

  • 2sinAcosB = sin(A + B) + sin(A - B)
  • 2cosAsinB = sin(A + B) - sin(A - B)
  • 2cosacosB = cos(A + B) + cos(A - B)

  • sin(90 - A) = cosA
  • sin(90 + A) = cosA
  • sin(180 + A) = -sinA
  • sin(270 + a) = -cosA
  • sin(360 + A) = sinA
